= Pragmas

{wp:Pragmas|Directive_(programming)} change various parameters (configuration) of CM processing. A pragma starts with ~@~ in the first position, followed by the pragma type and then a value or values.

== Saving and restoring configuration state

The current configuration can be stored (on a stack) by the pragma ~push~:
~~~
@push
~~~
and restored by the pragma ~pop~:
~~~
@pop
~~~

== Pragma ~chr~
Special characters that trigger CM processing can be changed with the pragma ~@chr~. For example, ~#~ as a comment (if used in the first position):

~~~
# This is a comment.
@push
# still a comment
@chr cmt ;
# not a comment
; because comments now start with a semicolon
: not with a colon
@chr cmt :
: and now with a colon
; not with a semicolon
@pop
# initial comment setting
~~~
makes {>}
---.box
# This is a comment.
@push
# still a comment
@chr cmt ;
# not a comment
; because comments now start with a semicolon
: not with a colon
@chr cmt :
: and now with a colon
; not with a semicolon
@pop
# initial comment setting
---

Comments can be completely disabled:
~~~
@chr cmt
: a
; b
# c
~~~
makes {>}
---.box
@chr cmt
: a
; b
# c
---

==== pragma ~chr~ values
The following table lists the possible CM characters that can be changed with the pragma ~chr~:
--- table.border
[value] [initially]      [purpose] 
(~cmt~)     (#)     (comment)
(~h~)       (=)     (header)
(~hr~)      (-)     (horizontal line)
(~p~)       (\.)     (paragraph)
(~ul~)      (-)     (unnumbered list)
(~ol~)      (\*)     (numbered list)
(~pre~)     (\~)     (preformatted text)
(~sec~)     (-)     (fenced section)
(~cls~)     (-)     (class attribute)
(~hook~)    (\{\|\}) (hook)
(~esc~)     (\\)    (escape)
(~b~)       ()      (bold shortcut)
(~em~)      ()      (emphasized shortcut)
(~u~)       ()      (underlined shortcut)
(~sup~)     ()      (superscript shortcut)
(~sub~)     ()      (subscript shortcut)
(~code~)    (\~)     (code \(monospaced text\))
(~macro~)   (\$)    (macro expansion)
(~th~)      (\[\])  (table header)
(~td~)      (\(\))  (table cell)
(~squo~)    ()      (typographical single quotes)
(~dquo~)    ()      (typographical double quotes)
---

Note: ~hook~, ~th~, and ~td~ take three, two, and two characters, respectively. The rest take one (or none).
